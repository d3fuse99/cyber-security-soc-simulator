Sentinel SOC Dashboard

Cyber Security Operations Center (SOC) simulation interface.

Features

* Live network log terminal
* Global attack feed (real-time)
* Threat injection simulator
* Incident analysis report
* Countermeasures log
* Network load analytics

Technologies

* HTML
* CSS
* JavaScript

Description

Sentinel SOC is a simulation of a Security Operations Center dashboard.
It visualizes network traffic, simulated cyber attacks, and automated defense responses in real time.

The project demonstrates UI design for cybersecurity monitoring systems.
